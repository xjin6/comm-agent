# Paper Machine

An AI-powered academic research toolkit — from data collection to paper writing.

Paper Machine is a modular suite of tools designed to assist researchers throughout the entire
academic workflow: scraping data, cleaning datasets, running analyses, generating visualizations,
and drafting papers.

## Modules

| Module | Status | Description |
|--------|--------|-------------|
| [skill-weibo-topic-scraper](./skill-weibo-topic-scraper/) | v0.1.0 | Scrape Sina Weibo topic posts, comments, and user profiles for social media research |
| *more coming soon* | | |

## Vision

Paper Machine aims to become an end-to-end research agent that helps academics go from a research
question to a publishable paper — handling the tedious parts so researchers can focus on thinking.
